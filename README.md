# Talkio

Walkie talkie satellite comms 🛰️

Features:
- ⚡ Real-time voice chat with low latency
- 🔒 Secure authentication (JWT, bcrypt)
- 🏠 Multi-room support with admin controls
- 📡 MQTT-based floor control for push-to-talk
- 🌐 Cross-platform: Electron desktop & Capacitor mobile
- 🎨 Modern UI with React, TailwindCSS, and Vite

Tech Stack:
- Frontend: React, Vite, TailwindCSS
- Backend: Express, MongoDB, Mongoose
- Desktop: Electron
- Mobile: Capacitor
- Real-time: MQTT
- Auth: JWT, bcrypt
